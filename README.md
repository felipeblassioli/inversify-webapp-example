# Inversify WebApp (Controller → Service → Repository)

Minimal Express + InversifyJS app with IoC wiring, plus Jest + Supertest tests.

## Install
```bash
npm install
```

## Run (dev)
```bash
npm run dev
# http://localhost:3000/api
```

## Build + Start
```bash
npm run build
npm start
```

## Test
```bash
npm test
```
